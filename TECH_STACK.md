# Tech Stack

Here is a simple breakdown of the main technologies used in this application:

*   **Next.js**: The main framework used to build the structure and pages of the website, making it fast and organized.
*   **React**: A library for building the interactive user interface components you see, like buttons, forms, and menus.
*   **TypeScript**: A version of JavaScript that helps prevent coding errors by checking the code for mistakes as it's written.
*   **Tailwind CSS**: A tool for styling the application's appearance, allowing for rapid and consistent design.
*   **ShadCN UI**: A collection of pre-designed, accessible interface components that speeds up development.
*   **Firebase**: A backend platform from Google that handles user sign-ups, logins, and the database.
*   **Genkit**: The AI toolkit from Google used to power the intelligent features, like legal draft generation.
*   **Zod**: A small library that helps ensure data flowing through the app is in the correct format, preventing bugs.
*   **Lucide React**: The library that provides the clean and simple icons used throughout the user interface.
